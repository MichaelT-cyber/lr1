Система пропусків (ЛР1)
  Це односторінковий веб-додаток для обліку пропусків у комп’ютерний клас.
Проєкт виконано в рамках лабораторної роботи з дисципліни «Основи алгоритмізації та програмування».
Реалізований функціонал:
Створення нового пропуску;
Редагування записів через форму;
Видалення записів із журналу;
Пошук записів за ім’ям студента;
Сортування за датою або ім’ям;
Збереження даних у localStorage (дані не зникають після перезавантаження сторінки).
Особливості реалізації:
Використовується внутрішній стан програми (state);
Кожен запис має унікальний ідентифікатор;
Реалізовано делегування подій для таблиці;
Таблиця оновлюється динамічно без перезавантаження сторінки.

Структура проєкту
frontend/
  index.html
  styles.css
  app.js

Запуск
1. Відкрити папку з проєктом у VS Code.
2. Запустити файл index.html через Live Server.
3. Додати тестові записи для перевірки роботи.
