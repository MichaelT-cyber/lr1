# Система пропусків — Backend

Backend-застосунок для системи пропусків у комп’ютерний клас.  
Проєкт реалізовано на Express + SQLite.

## Технології

- Node.js
- Express
- SQLite
- TypeScript
- tsx

## Функціонал

Система підтримує:
- роботу з користувачами;
- роботу з пропусками;
- роботу з причинами;
- журнал логів.

Реалізовано REST API для CRUD-операцій.

## Безпека

У проєкті реалізовано:
- захист від SQL Injection через параметризовані SQL-запити;
- захист від IDOR через перевірку userId;
- захист від XSS через безпечне відображення даних;
- базові security headers;
- централізовану обробку помилок.
